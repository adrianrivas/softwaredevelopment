En js tenemos 3 tipos de variables
Las var, let y las const

Las var tienen un caracter global
y puede accederse a ella desde cualquier
lugar del script

Las variables let tienen un caracter
de bloque y solo pueden usarse en el
bloque donde son declaradas

Las variables const son variables constantes
a las cuales no puede cambiarsele
el valor.

La desestructuración me permite crear varias 
variables simultaneamente y al mismo tiempo
darles valor al recorrer un array o un objeto

Para correr modulos en Javascript, necesito 
correr mi sitio en un servidor web, para ello
descargo la extensión live server en visual
studio code y listo.

Al exportar por defecto varias funciones de otro modulo,
debo declarar un objeto en el modulo donde las voy a utilizar
para poder utilizar mis funciones exportadas
